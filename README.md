# UP3D
Unstructured-mesh full-potential solver for 3D flows.

## Docs

- [docs/design.md](docs/design.md): design reference for equations, numerics, architecture, and validation targets.
- [docs/roadmap.md](docs/roadmap.md): active implementation roadmap, gates, and progress ledger.
- [docs/agent-rules.md](docs/agent-rules.md): short working rules for the coding agent.

## Naming

The current naming is semantic: `design.md` is the reference document, while `roadmap.md` is the live execution plan, and `agent-rules.md` holds the short working rules.
