# NACA0012 M∞ = 0.5 subsonic-compressible reference (gate G3.2)

Reference `cl(alpha)` and `Cp(x/c)` for the closed-TE NACA0012 at
M∞ = 0.5 in subcritical inviscid flow, generated by
`generate_reference.py` from the SAME converged Hess–Smith panel solution
as the P2 incompressible reference
(`../naca0012_incompressible/`, N = 800 panels, identical closed-TE
geometry from `pyfp3d.meshgen.planar.naca0012_coordinates`), corrected
for compressibility two independent ways and integrated over the same
panel geometry:

- **Prandtl–Glauert**: Cp/β, β = √(1−M²). First-order; a known *lower*
  bound on suction-side amplification for a 12%-thick section.
- **Kármán–Tsien**: Cp/(β + (M²/(1+β))·Cp/2) (Anderson, *Modern
  Compressible Flow*, ch. 9). Hodograph-based; more accurate pointwise,
  slightly *over*-weights the integrated amplification.

## Files

- `cl_reference.csv` — alpha = 0, 2 deg: cl incompressible / PG / KT.
  Gate values at **alpha = 2°: cl_PG = 0.278774, cl_KT = 0.291861**;
  the G3.2 reference is the two-correction midpoint
  **cl_ref = 0.285318** with the half-spread ±2.3% recorded as the
  correction-model uncertainty (the exact subcritical full-potential
  answer is bracketed by the two corrections).
- `cp_alpha2_m05.csv` — panel-control-point Cp at alpha = 2°:
  incompressible / PG / KT columns, tagged upper/lower (V3.2 overlay).

- Underlying incompressible panel solution is the P2-verified one
