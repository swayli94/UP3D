# NACA0012 incompressible inviscid reference (gate G2.3)

Reference `cl(alpha)` and `Cp(x/c)` for the closed-TE NACA0012 in
incompressible inviscid 2D flow, generated by `generate_panel_reference.py`
(Hess–Smith panel method: constant-strength source per panel + single
constant vortex strength + Kutta closure) — an independent method
(boundary-element) from the FEM solver it gates, on the SAME closed-TE
coordinate set (`pyfp3d.meshgen.planar.naca0012_coordinates`, last
thickness coefficient −0.1036) so G2.3 compares method vs method on
identical geometry.

## Files

- `cl_reference.csv` — cl at alpha = 0, 2, 4, 6 deg, N = 800 panels
  (cosine-clustered). G2.3 gate value: **cl(4°) = 0.482556**.
- `cp_alpha4.csv` — Cp at panel control points, alpha = 4°, N = 800,
  tagged upper/lower (for the V2.3 overlay plot).
- `convergence.csv` — cl vs panel count (N = 100/200/400/800) for both
  the Cp-integrated and the Kutta–Joukowski value.

## Verification trail (2026-07-06)

- Panel-count convergence: cl(4°) = 0.481939 → 0.482556 over N = 100 → 800,
  monotone, last increment 8e-5 (~0.02%).
- Two independent lift routes agree: Cp integration 0.482556 vs
  Kutta–Joukowski −2·tau·perimeter = 0.483011 at N = 800 (0.09%).
- alpha = 0 gives cl = 0 to machine precision (symmetric section).
- Lift slope ≈ 6.91/rad vs the classical thickness-corrected estimate
  2·pi·(1 + 0.77·t/c) = 6.90/rad for t/c = 0.12.
