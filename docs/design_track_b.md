## 1. 动机重估（2026-07-11 视角）

DN1 成文于 2026-07-07（P4 刚关闭、P8 Newton 尚未存在）。当时列出的两类收益，
在 P8/P10 关闭后地位已经不同，必须重估：

### 1.1 已过时的动机：消除 Γ 外层循环的**效率**收益

DN1 §2.5/§8-B4 的核心效率论点是：罚函数/隐式 Kutta 去掉 Γ secant 外层后,
P4 的三层嵌套（Mach × Γ secant × Picard）变两层，预期 medium 16m39s → 8–10min。

**这一动机现已基本失效。** P8 已交付全耦合 (φ_red, Γ) Newton
（`solve/newton.py`，roadmap P8 台账）：Γ 是耦合未知量、每步精确 δΓ 消元
（Woodbury 秩-n_st 修正），不存在 Γ 外层迭代；实测 NACA medium ~100 s
（对比 Picard 16m39s 到一个非解状态）、M6 medium 140–249 s。Track B 在求解
速度上对 Newton 路径**没有可预期的显著增益**——它去掉的 Γ-secant 嵌套已经被
Newton 去掉了。B3/B6 gate 里 "Picard count ≤ current / 减少 30–50%" 的表述
只对 Picard-vs-Picard 的 A/B 有意义（见 §7）。

### 1.2 仍然成立的动机：网格/几何工作流能力（★用户裁决 2026-07-11：此即 Track B 的目的）

以下收益与求解器速度无关，P8 之后依然成立，且是 Track B 的**真正定位**——
**该定位已由用户确认，是 §4 差异裁决与 gate 设计的最高准绳**：

1. **网格生成不再嵌入尾迹面**。当前 `occ.fragment`+`mesh.embed`+`wake_cut.py`
   链条是 roadmap §12 记录的 #1 工程风险（M1 交付时的 fragment/embed 陷阱、
   尖端自由边、探针跨站共享等都源于此）。
2. **改攻角不重新生成网格**——气弹/优化迭代、α 扫掠的前提（论文 §1.1 图 1.2–1.3
   正是用这个场景论证嵌入式方法的必要性）。
3. **M2 翼身组合体**。尾迹–机身交线是 conforming 面处理得最差的场景
   （roadmap M2 行注记）；论文 §4.9 用 CRM WBH/WBNPH 证明嵌入式尾迹直接
   跨过机身交线捕捉 [φ] 跳跃（图 4.53），无需任何专门的交线网格处理。
4. **钝尾缘**。CRM 机翼是钝尾缘（最小工艺厚度约束，论文 §4.9.1），论文照常
   求解——尾迹条件在钝尾缘处直接施加，Cp 在 TE 处相对 RANS 高估但有界
   （§4.9.3）。conforming 路径对钝尾缘没有清晰的锚定方案。
5. **结构性消除 P5 的 st133 类故障**。conforming 路径的 per-station Kutta
   探针在后掠 TE 上跨站共享节点（P5 记录的未修鲁棒性隐患，st133 单站失稳的
   根因）。嵌入式路径没有站、没有探针——尾迹条件是逐单元最小二乘残差，
   该故障类在结构上不存在。

### 1.3 Track B **不解决**什么（边界划清）

- **不动精度地板**：0.019 的 M6 升力缺口归因于平坦 P1 壁面（P9 正在判别，
  修复属 P11 曲面壁元）。Track B 只改尾迹表示，壁面离散完全不动。
- **不动 FP 折叠区模型极限**：非唯一性折叠（NACA medium M≈0.79）是全势方程
  本身的性质，与尾迹表示无关。折叠区纪律（单网格回归锁、保留 Mach ramp、
  禁松容差）原样适用于 B 路径。
- **不是 V6<1% 的路线**：V6 残余是尖 TE/LE 的 P1 壁面梯度地板（G5.2 重定义），
  与尾迹表示无关。

## 2. López 论文的嵌入式尾迹方法（精确摘要）

论文第 3 章是 Track B 的直接原型：**body-fitted 几何 + 嵌入式（隐式）尾迹**,
即 Davari 2019 架构的跨声速 3D 工程化。与 Núñez 2022（几何也嵌入）不同，
几何仍由网格贴体描述——与 pyFP3D 的处境完全一致。

### 2.1 尾迹单元与辅助 DOF（§3.5，Eq. 3.33–3.34）

尾迹几何 = 从 TE 沿来流方向延伸到出口的直线（2D）/直纹面（3D），**不在网格中
显式建模**。被尾迹面切到的单元称尾迹单元。尾迹单元的每个节点富集一个辅助
DOF φ_aux（**逐节点**，跨相邻尾迹单元共享——论文图 3.6 的上下"分离网格"仅为
可视化，实际只有一套网格），上/下侧的单元值按节点在尾迹哪一侧选取：

```text
φ_u^i = φ_i      若节点 i 在尾迹上方        φ_l^i = φ_aux^i  若节点 i 在尾迹上方
       = φ_aux^i 若节点 i 在尾迹下方               = φ_i      若节点 i 在尾迹下方
```

**行分配**（§3.5 末 + §3.5.4）：主 DOF 行 = 质量守恒（FP Galerkin 残差,
由节点所在侧的单元拷贝装配）；辅助 DOF 行 = 尾迹边界条件（§2.2）。
这是一个**替换式**方程分配，不是罚函数叠加——总体矩阵因此非对称（见 §5.3）。

### 2.2 尾迹边界条件：线性化 + 最小二乘（§3.5.1–3.5.2）

物理条件：法向质量通量连续（Eq. 3.5）+ 压力相等（Eq. 3.6）。

**2D**（Eq. 3.35–3.42）：合成单个矢量方程 u_u − u_l = 0 于尾迹单元域 Ω_W,
最小二乘泛函 Π = ½∫(u_u−u_l)² dΩ_W，对 DOF 求导得残差/Jacobian。LS 弱式下
这等价于强制 [φ] 在尾迹带内（弱）常数——即 Γ 沿尾迹守恒。

**3D**（Eq. 3.43–3.51）——**只取两个标量分量,不是完整矢量**：

- g₁ = n̂·(u_u − u_l) = 0 （质量条件,用 ρ∞ 替代 ρ_u/ρ_l 线性化）
- g₂ = û∞·(u_u − u_l) = 0 （压力条件线性化：[φ] 沿来流片流线的齐次对流方程,
  即 ∇[φ]·û∞ = 0——跳跃沿流向常数）

Π = ½∫(g₁²+g₂²) dΩ_W。**展向切向分量刻意留空**：∂[φ]/∂(展向) 正是尾涡片
强度，3D 有限翼展必须允许非零。论文图 4.40–4.41（M6 尾迹 [φ] 云图 + TE/
Trefftz 面对比）验证了 g₂ 的对流性质被正确离散。

Jacobian 块（Eq. 3.48–3.51）形如 ∫ ∂N_u/∂x_b (n̂_b n̂_a + û∞_b û∞_a) ∂N_l/∂x_a,
**在 φ 中是线性/常数矩阵**——对 Newton 是一次装配、永不重建的块（§5.5）。

### 2.3 TE 节点处理（§3.5.4）

TE 节点属于物体：施加滑移条件（q=0 的自然 BC），其辅助 DOF **只用于断开上下
单元并实现质量守恒方程**，不写尾迹 BC。即 TE 的 φ_aux 是"下侧单元的普通势
DOF"。这与 P2 的来之不易的证据完全一致：**TE 节点必须复制**（连续统里
[φ](TE)=Γ 就是 Kutta 条件；不复制会在 TE 停一个 Γ²/h 发散的假点涡,
`mesh/wake_cut.py` docstring）。

> **⚠️ 以下段落已于 2026-07-12 撤回，见 §9.2。** g₂ 的线性化形式对
> **常数跳跃恒等于零**（单位分解 Σ_c ∇N_c = 0，实测残差 1.9e-16），
> **结构上不可能钉住 Γ**。论文方法章全文亦无 "Kutta" 一词。真正钉住 Γ 的是
> **TE aux 行（下侧质量守恒）这一个方程**——而它的控制体上下不对称，正是 B3
> 过环量 45% 的根因（新增阶段 B4）。原文保留如下作为记录：

~~**Kutta 条件没有任何显式罚项**（全文无 penalty Kutta；见 §4-D2）：TE 紧后方
尾迹单元上的 g₂（压力相等的线性化）就是离散 Kutta 条件，Γ 的正确值由它自洽
确定。论文 §4.1 的不可压 NACA0012 升力精度 ~O(0.1%)（vs 参考解）证明该隐式
施加是充分的。~~

### 2.4 落在尾迹面上的节点：ε 平移（§3.5.3）

节点恰好在尾迹面上 ⇒ 上下同值 ⇒ 跳跃归零、解错。处理：给这些节点赋一个正的
容差距离（如 ε=1e-6，量纲为长度）——等价于把尾迹面局部下移。方向任意但需
一致。TE 节点是此规则的例外（§2.3 的专门处理）。

### 2.5 小切割问题与全单元积分（§3.5.5）

首版实现（按几何切割的子域积分）在节点贴近尾迹面时病态：小切割子域 → Jacobian
条目 ~0（Eq. 3.52）；重算形函数导数则 ∝1/A → 条目爆炸（Eq. 3.53）。
**最终方案：尾迹 BC 积分（Eq. 3.37–3.42, 3.46–3.51）在整个单元上进行**
（full integration），条件数与 body-fitted 方法相当。对 P1 四面体这正好
落回 DN1 §3.3 已识别的简化：**切单元不需要几何切割积分，只是同一 (B_e, V_e)
用两套 DOF 编号装配两次**。代价是切割层 O(h) 体积的重叠一致性误差（沿测度零
面），被论文的网格收敛研究（图 4.17：cl 误差随 h 干净收敛至 <2%→<0.5%）证明
不伤收敛阶。

### 2.6 稳定化与延拓（与 pyFP3D 已一致的部分）

人工密度 ρ̃ = ρ − μs(ρ−ρ_up)、μs = μc·max(0, μ, μ_up)（Eq. 3.19–3.22）与
design.md §3.2 同族；等熵限速（§3.4，M²_max ~ O(3) 处冻结密度）对应 pyFP3D
的 q² 限幅器；Mach ramp（表 4.13：M∞ 0.5→0.84 共 8 级、M_crit=0.95、μc=2.0
固定，达标后 μc 2.0→1.6 四级退火）正是 P8 NEWTON_M6_RECIPE / G10.3
"保留 ramp" 裁决所对齐的先例。**Track B 不改这些**——B6 直接继承
`damping_theta`/continuation/upwind 全套。

## 3. Track B 数值方案（吸收论文证据后的定式）

方案骨架沿 DN1 方案 B，但按论文证据修订（修订点在 §4 逐条对照）：

1. **尾迹表示**：`WakeLevelSet` 有符号距离；直尾迹 = 从 **TE 线**（3D 是折线/
   曲线，不是单点——见 §4-D9）沿来流方向的直纹半面。多尾迹 = 多实例。
2. **切单元识别**：四面体 4 节点 φ_wake 符号不一致 ⇒ 切单元；节点侧标记
   ± ；**|φ_wake| < ε 的节点强制归 + 侧**（§4-D4）；TE 节点例外处理。
3. **富集**：逐节点辅助 DOF（全局去重共享），N_ext ≈ 切单元节点数
   （M6 medium 量级 10³–10⁴，相对 63k 主 DOF 可忽略）。
4. **装配**：切单元同一 (B_e, V_e) 全体积装配两次（上/下两套 DOF 编号）,
   无几何切割积分（论文 full integration，§2.5）。
5. **尾迹 BC**：2.5D/2D 用矢量式 u_u−u_l=0 LS；**3D 用且仅用 g₁+g₂ 两分量
   LS**（§4-D1）。行分配：主 DOF 行=质量守恒（本侧），辅助 DOF 行=尾迹 BC。
6. **Kutta**：**基线 = López 隐式**（TE 节点滑移+复制，TE 后方尾迹单元的 g₂
   即 Kutta），`wake/kutta.py` 罚函数降级为备用诊断路径（§4-D2）。
7. **Γ**：不再是未知量/参数。后处理提取 Γ(z) = φ_main − φ_aux 于 TE 节点
   （按侧向约定校正符号），喂给 cl_KJ 与（若保留的）远场涡修正。
8. **线性求解**：矩阵非对称 ⇒ GMRES + AMG(主块)（P8 起 `solve/linear.py`
   已有 GMRES/ILU 基建）；对称化备选方案见 §5.3。
9. **跨声速**：上游走 walk 的邻接限制在同侧虚拟单元图上（§5.2）；
   `damping_theta`、Mach ramp、q² 限幅原样继承。

### 按已裁决的定位重排优先级（工作流能力 > 求解效率）

定位裁决（§1.2）不改变 D1–D11 的技术裁决本身（它们是正确性/鲁棒性判断,
与快慢无关），改变的是**优先级排序与验收归属**：

- **工作流目标的阻断项（做错则定位落空）**：
  - **D1 + D9**——3D 尾迹 BC 分量选择与 TE 线直纹面 API 是 M2 翼身组合体
    和一切 3D 翼的前提;M2 需要机翼+平尾多条 TE 线、多 level-set 实例。
    这两条错了,Track B 只剩一个 2.5D 玩具。
  - **D4 + D6**——α 扫掠会让尾迹面以**任意角度**扫过固定网格,扫掠序列中
    必然出现节点贴面与近退化切割:ε 归侧和 full integration 不是数值洁癖,
    是"扫 α 不重网格"这一能力本身的鲁棒性前提。ε 取 c·h_local、归侧规则
    确定性（固定 + 侧）,保证解在 α 扫掠中不因切割拓扑抖动而跳变。
  - **D8**——走廊尺寸场必须按 α 范围设计（楔形/扇形包络）,否则扫掠中
    某些 α 的尾迹落进粗网格区,[φ] 捕捉质量随 α 突变。
- **工作流红利（顺手拿到,不另立目标）**：D2 隐式 Kutta 恰好在钝尾缘 CRM
  上验证过——钝尾缘正是工作流目标之一;罚函数保留为钝尾缘病例的备用加强。
  逐单元尾迹条件结构性消除 st133 类探针故障（§1.2 第 5 条）。
- **正确性管道（与定位无关,但必须对）**：D3/D5/D10/D11。
- **测量决策（不预判）**：D7 远场升级为独立对比阶段 B5（用户裁决,§5.4）。
  效率类指标全部降为**非回归守卫**——B3 的 "Picard count ≤ current" 只防
  成本爆炸,不作为收益主张（§1.1）。

### D1 展开：为什么 Núñez 矢量式在 3D 是错的

ρ⁺∇Φ⁺ = ρ⁻∇Φ⁻ 是**三分量**方程。跨尾迹 ρ 连续（压力相等 ⇒ 等熵密度相等）,
故它蕴含 ∇Φ⁺−∇Φ⁻ = 0 的**展向分量也为零**，即 ∂[φ]/∂(展向) = 0——这恰好是
尾涡片强度，有限翼展下必须非零（椭圆环量分布的 dΓ/dz 就住在这里）。Núñez
2022 是嵌入式**翼型**（2D）文章，矢量式在 2D 正确；DN1 把它推广到 3D 时引入
了这个过约束。López 的 3D 式（Eq. 3.43–3.44）刻意只取法向+流向两分量,
展向留空，且经 M6/CRM 全套 3D 跨声速验证。**B2/B3 起（2.5D 单层网格）就应
直接实现 g₁+g₂ 式**——2.5D 网格上展向梯度本就 ~0，两式数值等价，但代码只写
一次、直通 3D；矢量式仅作为 2D 交叉验证的诊断选项。

线性化的适用性：g₁ 的 ρ∞ 替换与 g₂ 的 û∞ 替换（精确式是均流速度
(u⁺+u⁻)/2 对流跳跃）都假设尾迹附近扰动小。论文在 M0.84/M0.85、激波、钝尾缘
下验证通过；直尾迹本就与来流对齐（design.md §4），线性化与直尾迹假设同阶。
若未来 B 后续阶段发现强激波掠过尾迹的病例，非线性式可作为逐项开关重进,
但当前无证据需要。

### D2 展开：Kutta 隐式化去掉一个调参风险

DN1 风险表第 1 行（k_kutta 敏感性,中等严重度）随罚函数一起消失。论文的隐式
方案在 body-fitted 几何上成立的原因：TE 节点真实存在于网格中（Núñez 需要
Algorithm 1 与罚函数,是因为嵌入式几何里 TE 不是网格实体）。pyFP3D 是
body-fitted——与论文同侧。**保留 `wake/kutta.py` 作为可选诊断**（万一隐式
Kutta 在粗网格上环量偏软,罚函数是现成的加强手段），但 B3 gate 以隐式路径
为准。

### 5.2 upwind 的真实改动（修订 DN1 §4.3 的错误论断）

人工密度按单元操作没错，但**切单元有两个速度状态**（上/下两套 DOF 给出两个
∇φ、两个 q²、两个 ρ）。TE/翼尖角部的切单元可以超声速（P5 记录的 tip-TE
角部过冲），所以不能假设切单元永远亚声速。需要：

- `rho_tilde` 对切单元逐侧求值：(e, side) 虚拟单元；
- 上游走 walk 的面邻接图对虚拟单元**限制在同侧**（尾迹面是滑移线,上游影响域
  不跨越它——物理上密度信息不穿过切向间断）；非切单元照旧。
- P7 冻结选择灵敏度 (s_e, s_u) 同样逐侧扩展（B 路径 Newton 时才需要）。

实现量可控：`UpwindOperator` 的 SoA 数组加一个 side 维度只在切单元上激活,
非切路径 bit 不变（结构守卫测试照抄 P8 模式）。

### 5.3 线性系统的对称性代价与求解器选择

替换式行分配（主行=FP、辅行=尾迹 LS）使矩阵**结构性非对称**（A[main,aux]
来自 FP 残差、A[aux,main] 来自 LS 条件,算子不同）。这推翻 DN1 §7 "整体 AMG
(-CG)" 的默认设想。选项：

- **基线：GMRES + AMG(主块)/ILU 预处理**——P8 已交付 `solve_gmres`、
  `build_ilu_preconditioner`,且 P8 的教训（精确 Jacobian 在超声速区本就非
  对称、CG 不可用）意味着 Newton 路径迟早非对称,Picard 路径提前非对称并不
  新增基建。硬规则 4（SciPy/PyAMG）不破。
- 备选（若 GMRES 在大网格 Picard 上失速）：Núñez 的叠加式行分配（LS 罚项
  加到双侧行上、FP 残差双侧保留）给出对称系统,可回 CG——但引入罚权标定,
  且论文没有走这条路。**先测基线,不预建备选。**

条件数风险由 full integration 压掉（§2.5）；AMG 聚合把 N_ext 个辅助 DOF
当普通节点处理即可（几何上与宿主节点重合,平滑聚合不敏感）。

**★ 落地（B11, 2026-07-14）。** 基线已实装（`solve/picard_ls.py`,
`solve/newton_ls.py` 的 `precond=None|"ilu"|"amg"`；None=原 `spsolve` 逐位
默认；`solve_multivalued_transonic` 经 `**kwargs` 继承）。关键实现细节修正了
本节"当普通节点处理即可"的一处过简：

- **★ ILU 是落地的有效逃逸路径;AMG(主块) 实测在 wake_ls 升力算子上失速。**
  `"ilu"`（融合 A_free 上的 spilu）分解真实融合矩阵,coarse M0.5 升力 434 步收敛、
  |Δγ|<1e-8、逐 outer 热启动——**这条才是解除 splu 墙的路线**。
  `_amg_surrogate_preconditioner` 在 `op.assemble_matrix(rho_own)`（单值 Picard
  刚度,conforming Newton "AMG 建在 Term-1 SPD 块而非融合 Jacobian" 的 LS 对应物）
  之上,为每个辅助 DOF a 与其宿主 h（`cm.ext_dof_of_node`）加 SPD 弹簧
  `k_a·(e_h−e_a)(e_h−e_a)ᵀ`,`k_a=|A_fused[a,a]|`（纯块对角单点会让 SA 把每个
  aux 单独成簇；弹簧使其与宿主同簇）。**但实测:此代理只对 `continuity` 闭合
  （Laplace/B2,aux 行=焊接 aux=main）收敛;对 `wake_ls` 闭合的升力/跨声速/Newton
  算子失速**——那里的 aux 行是 g₁+g₂ 尾迹 LS + 非线性 TE Kutta（对流型,非 SPD
  弹簧）,代理无法建模,GMRES 顶着 restart 上限空转（coarse M0.5 升力:γ 0.0033 vs
  0.139,80/80 outer 全 stall,455 s vs ILU 2.7 s）。⇒ **AMG 保留为 Laplace 情形
  + §5.3 记录旋钮,升力路径落地用 ILU。**
- Núñez 对称行分配（备选）**仍不预建**——它才是真正能恢复 AMG 适用性的路线,留作
  ILU 填充在 fine 尺度成为瓶颈时的下一步。
- Picard 外层热启动 GMRES（上一 outer 的自由 DOF 解）,每 `amg_rebuild_every`

- `linear_rtol=1e-10`：Picard 判据在 1e-6/1e-7、newton_ls gate 在 1e-10,故
  1e-10 相对线性解不扰动不动点/Newton 终点（测试锁 |Δγ|<1e-8）。
- **lagged-LU（`direct_refactor_every`,solve/newton.py）移植进 newton_ls 不在
  B11 范围**（§11.5 gap 2 的记录项）：B11 交付迭代逃逸,非直接重用；若 M6 证据

**★ 附记（B12+B13, 2026-07-14）——迭代逃逸的实测边界与 medium 落地路线的更替。**

### 5.4 远场：两个自洽选项（D7 → 独立对比阶段 B5,用户裁决 2026-07-11）

conforming 路径的远场是 Dirichlet φ∞+Γ(z)-taper 涡修正（P5 修复件）。
B 路径上 Γ 不再是先验参数：

- **选项 a（保守,推荐先做）**：保留球形远场+涡修正,每次 Picard 外迭代后从
  当前解提取 Γ(z)（TE 的 [φ]）刷新 Dirichlet RHS——RHS-only,与 P3 密度
  更新同节奏,不构成新的外层循环。**切单元触远场时,主/辅 DOF 都要给 Dirichlet
  值**（各自侧的涡修正值——涡修正本身跨尾迹多值,conforming 路径的 master/
  slave 赋值逻辑平移过来）;DN1 "Dirichlet 只作用主 DOF" 在远场切节点上不完整。
- **选项 b（López 式）**：出口 Neumann q = n̂·(ρ∞u∞),无涡修正,域半径换
  尺寸场补偿。论文 §4.1.4 的域尺寸研究（δcl 随 L 收敛）是其证据,但 pyFP3D
  的 15c 球域是按涡修正标定的（design.md §5）,换 Neumann 需要重标定域尺寸。

**裁决（2026-07-11）：两个选项都实现,以独立阶段 B5 实测对比**。B3 本身
用选项 a（保持与 conforming 路径同域同网格的严格 A/B）;B5 在亚声速
NACA + M6 coarse 上对比 a/b（选项 b 先按论文 §4.1.4 的域尺寸研究方法重标定
域半径/尺寸场）,以测量证据选定 B6 之后的默认。注意选项 b 对工作流定位更有
吸引力——无 Γ 提取回馈、α 扫掠循环最简——但不预判,B5 说了算。

**★ B5 结论（2026-07-12,已关闭；接口 `solve_multivalued_lifting(farfield=
"vortex"|"neumann"|"freestream")`,默认 `"vortex"`）：选项 a（Dirichlet+涡）
保持默认。** 按论文 §4.1.4 的域尺寸研究方法,在 NACA 双网格族（M0 嵌入 +
M3 无尾迹）上做 R∈{15,30,60,120}c 扫掠（coarse, M0.5 α2°）：
- **选项 a 域无关**：Γ 在 15→120c 上偏差仅 0.45%(M0)/1.09%(M3),15c 处与
  conforming 差 0.25% ⇒ 15c 紧凑域 + 涡修正就够。
- **选项 b 截断 O(Γ/R)** 点涡远场尾：15c 处 −4.07%,R 每翻倍误差减半
  （30c −2.0%, 60c −0.99%, 120c −0.50%）⇒ 要进 B3 的 ±2% 带需 **R≥~30c**、

- freestream（全场 Dirichlet 自由流、无涡）每个 R 都最差,且在紧凑 15c M0

两网格族结论逐位一致。O(Γ/R) 截断与几何无关（3D 机翼截断同一马蹄涡尾），
故此结论同样定了 M6 B 路径的远场默认。**M6 腿并入 B7**（用户裁决

### 5.5 Newton 再推导（post-B6,继承 roadmap 排序护栏；2026-07-12 Track-B 重编号后）

P8 Newton 落在 conforming 尾迹上（Γ 消元、H 列、远场涡列 B、Kutta 行 K）。
B 路径 Newton 的结构**更简单**：

- 尾迹 LS 块（Eq. 3.48–3.51）在 φ 中线性 ⇒ Jacobian 常数块,一次装配;
- 无 Γ DOF ⇒ 无 δΓ 消元、无 Woodbury、无 `test_gamma_column_fd` 类陷阱
  （若远场走 §5.4 选项 a,Γ(z) 对 φ 的依赖仍产生一个低秩远场列——结构与
  P8 的 B 列同型,FD 守卫模式照抄;走选项 b 则彻底消失）;
- Terms 1–3（`kernels/jacobian.py`）经 DOF 间接层复用：切单元的 Term 1/2
  按侧装配两次,Term 3 的 COO 活动集加 side 维;
- 冻结/refresh/直接步全套 N5 机械不动（它们作用于上游选择与线性代数层,
  与尾迹表示正交）;
- 系统本就走 GMRES/splu,非对称无新增代价。

论文表 4.9（含尾迹 LS 块的 Newton 二次收敛,残差 4e-3→4e-9）是该结构可达
二次收敛的直接先例。**排序护栏不变**：B 路径 Newton 是 post-B6 的再推导,
不与 conforming Newton 并行设计。

> **mixed-side plain 元素**——未被切割、但四个节点跨越尾迹 level set（越过展向

- **(a) 带嵌入尾迹面的现有网格（M0/M1,"C 型网格"类比）**——尾迹面在网格
  拓扑中,所有尾迹面节点恰好落在 level-set 零面上。作用有二：其一,是 ε
  归侧规则（D4）的**规模化压力测试**（整片节点全走 ε 分支,任何不确定性
  立刻放大成 census 抖动）;其二,与 conforming 路径**同网格严格 A/B**——
  逐站 Γ(z)、cl、残差史都在同一离散上可比,是 B3/B4 验收的锚。
- **(b) 无尾迹面网格（M3 新增,"O 型网格"类比）**——网格生成完全不知道
  尾迹存在,切割是**一般位置**的（任意角度穿过任意单元）:这才是 Track B
  工作流定位的真实交付形态（免嵌入、α 扫掠）。没有 conforming 对照,
  验收对 (a) 的 B 路径结果与外部参考（PG/KT 区间等）比。
  M3 = M0 同管线去掉 embed + **尾迹走廊楔形尺寸场**（D8:无尾迹面就没有
  天然加密,走廊须覆盖 α 扫掠范围,否则 [φ] 捕捉质量随 α 突变）;
  **M4 = M1(ONERA M6) 同管线去掉 fragment+embed**（弦平面片只喂 Distance
  尺寸场）。

**M4 的走廊决策（2026-07-11 实测后定）**：3D 不用 M3 的楔形扫掠走廊——
楔体积随展长增长,±3° 包络在 M6 coarse 上要 ~4× 单元数,不成比例。M4 直接
复用 M1 的走廊尺寸场（同 h_wake、同阈值,片在同一弦平面）：单元数 coarse
50,605 / medium 329,645,对 M1 的 55.5k/350.7k 只差 6–9%——**这正是 B7
对 P5/P8 基线做受控 A/B 所需要的**。3D 的 α 重指向因此限于走廊已解析的
近标称带;宽包络的"扫 α 不重网格"演示放在便宜的 2.5D M3 上。

**3D 才暴露的两个机制（B1 实测,2026-07-11）**：
1. **斜交展向坐标（必须）**：后掠 TE 的展向轴与尾迹方向**不正交**,若用正交
   投影求展向坐标 q,远下游点的 x 位移会漏进 q,把点推到"翼尖之外"——实测
   在 M6 coarse 上错误裁掉真切集的 ~60%。正解是在 (v_span, d̂, n̂) **斜交基**
   里解 2×2（n̂ 与两者正交,s 仍是正交分量）。已回归钉死。
2. **展向裁剪（必须）**：尾迹面到翼尖为止,穿越点须满足 0 ≤ q ≤ span_length。
   否则 level-set 会把翼尖外的**尾迹平面延伸**也切开——这正是 P5 在 conforming
   路径上修掉的远场分支射线伪迹的 level-set 版本。conforming 路径靠自由边规则
   （尖端节点单值,Γ(tip)=0）拿到同一语义。

两类网格的差异本身就是测量对象：同一 level-set 路径在 (a)（切面与单元面
重合的退化切割）与 (b)（一般切割）上的 cl/Γ(z) 差应落在离散误差带内——
若不然,说明 ε 归侧或 full integration 的某个环节对切割几何敏感,这正是
α 扫掠鲁棒性风险的早期探测器。**已测的一个结构性差异（M6,B1）**：
level-set 切集是 conforming minus-star 的**严格超集**（缺失 0,多 2.9%）,
多出的全部是**翼尖边跨越单元**——嵌入式方法不要求尾迹面的**边**（尖端涡线）
贴合单元边,故这些单元被部分切割。这是预期行为（Γ→0 于尖端,跳跃在此收敛到
零）,B2/B3 需要在装配与 Γ 提取时按此语义处理,不是缺陷。

## 9. TE 控制体与"隐式 Kutta"的离散——**已解决（B4，2026-07-12）**

> 状态：**已解决**。§9.1–§9.4 是问题诊断（保留为证据链），**§9.7 是最终方案**：
> 非线性 TE 压力相等（Bernoulli）Kutta，速度在 **TE 的壁面邻接上/下控制体**上恢复。
> 实测 Γ 与 conforming **同网格 <1%**（M=0 与 M0.5 皆然），V3 的 cl 落入 [PG,KT]。
> 测试 = `tests/test_b4_te_control_volume.py`（机理+gate）与
> `tests/test_b3_lifting.py`（V3/双网格）。

### 9.1 症状

B3 的多值升力解（隐式 Kutta，Γ 自涌现）在 NACA0012 α=2°、不可压上给出

| 网格 | coarse | medium | fine | conforming/薄翼真值 |
| --- | --- | --- | --- | --- |
| Γ | 0.2074 | 0.1760 | 0.1704 | **0.1200** |

**网格收敛**（单调下降并趋于 ~0.168）⇒ 这是**方法误差，不是离散误差**。
自洽性已验证：cl_KJ = 2Γ = 0.372 与逐侧映射后的 **cl_压力积分 0.374** 吻合，
即流场确实过环量约 45%，不是提取 Γ 的 bug。

### 9.2 ★ 关键结论一：尾迹 LS **结构上不可能**钉住 Γ

LS 残差（本文 §2.2 / 论文 Eq. 3.46–3.51）为 `R_a = V_e Σ_c P_ac δ_c`，
其中 `P_ac = (n̂·B_a)(n̂·B_c) + (û·B_a)(û·B_c)`。对**空间常数**跳跃
`δ_c ≡ Γ`：

```
Σ_c P_ac = (n̂·B_a)(n̂·Σ_c B_c) + (û·B_a)(û·Σ_c B_c) = 0
                       ↑ Σ_c B_c = Σ_c ∇N_c = ∇(Σ_c N_c) = ∇(1) = 0   （单位分解）
```

⇒ **LS 对任意常数跳跃恒等于零**，LS 块含"常数跳跃"零空间。数值确认：把真解
（main + `aux = main − side·Γ`，势流在割线上解析，故这是精确的另一侧延拓）代入
B3 方程，**LS 行残差 = 1.9e-16**（`test_wake_ls_has_constant_jump_nullspace`）。

**⇒ 本文 §2.3 与 §4-D2 的论断"TE 后方尾迹单元上的 g₂ 就是离散 Kutta 条件"
是错的，予以撤回。** 论文全文亦无任何显式 Kutta 施加（"Kutta"一词在方法章
未出现）。

### 9.3 ★ 关键结论二：Γ 实际上只由**一个方程**钉住——TE aux 行

自由度计数：LS 行（非 TE aux）在给定 main 场后决定 aux 场，**但差一个常数模**
（§9.2 的零空间）；该常数模就是 Γ。唯一不属于 LS 的尾迹行是 **TE 的 aux 行**
（论文 §3.5.4 的例外：TE aux 承载**下侧质量守恒**而非尾迹 BC）。

**⇒ 整个"隐式 Kutta"= TE 处下侧质量守恒这一个方程。** 它的物理含义是"双 TE
节点 + 两侧滑移 BC 阻断绕 TE 的绕流"。

制造解检验（真解代入 B3 方程的分行残差，medium）：

| 行类别 | max\|R\| | 判读 |
| --- | --- | --- |
| 尾迹 LS 行 | **1.9e-16** | 精确满足（§9.2） |
| bulk 主行 | 4.1e-3 (mean 2.8e-4) | 离散误差量级 |
| 壁面主行 | 6.6e-4 | 正常 |
| **TE 主行** | **2.9e-3** | 真解**不满足** |
| **TE aux 行** | **2.4e-3** | 真解**不满足** ⇒ 这就是错的那个方程 |

### 9.4 ★ 根因假设：TE 控制体上下不对称

TE 单元扇形普查（medium，`test_te_control_volume_is_asymmetric` + artifact）：

- **上侧扇形（plain "+"）：9** 个
- **下侧扇形（te_lower，TE 引用 aux）：6** 个
- **切单元（片穿过）：3** 个

对称翼型的 TE 本应上下对称。不对称直接来自 **ε 上移约定**：尾迹面上的节点
全归 "+"，于是**尾迹面正上方的一层单元是"纯上侧"（不切），而其下方的镜像层
却被切开**——可视化 artifact 左图把这一点画得很清楚（TE 下游 x>1.0025 处，
上方 3 个红点 vs 下方 3 个绿点）。

论文称 ε 移动方向"arbitrary"（§3.5.3），但它在 **TE 处显然不是中性的**——
论文自己在 §3.5.4 (p.57) 专门警告过一个相关后果：「Moving these nodes upwards
would result in **cutting all wake elements under the wake that are in touch
with the trailing edge**, which yields inaccurate results」。

**已按该警告修掉一个真 bug**（2026-07-12，`wake/cut_elements.py`）：TE 的 ε 上移
使**完全位于尾迹面下方、只碰到 TE 顶点**的单元产生边缘正穿越而被误判为切单元，
拿到**尾迹下方的虚假上侧拷贝**。修复后 6/6 下侧扇形归位，Γ 的性质从"网格无关
的错值 0.186"变为**收敛序列 0.207→0.176→0.170**（虚假拷贝原先把 Γ 钉死在一个
假的网格无关值上）。但残余 42% 说明**还有第二个不对称源**——即 §9.4 的层级不对称。

### 9.5 B4 待办（roadmap Track B B4）

重新推导 TE 处的控制体/方程，使离散 Kutta 一致。候选方向（**未裁决**）：

1. **对称化 TE 控制体**：在 TE 邻域改用对称的切割/归侧规则（例如 TE 邻域
   不做 ε 上移，或对 TE 扇形显式配平上下单元），使 TE aux 行的下侧控制体与
   TE 主行的上侧控制体镜像对称。
2. **显式 TE 压力相等（真 Kutta）**：既然 LS 的线性化 g₂ 结构上无法钉住 Γ
   （§9.2），改在 TE 处施加**非线性**压力相等 |∇φ_u|² = |∇φ_l|²（Bernoulli），
   它对 Γ 不是退化的。这与 D2 的"罚函数 Kutta 备用路线"是同族——D2 正是为
   这种情形保留的。
3. **域尺寸**：论文的域尺寸研究用 **10²–10⁷ 弦长**（p.73 表），pyFP3D 是
   **15c + 涡修正**。已实测**排除**其为主因（用 conforming 真解的远场轨迹作
   Dirichlet，内部仍给 0.1721），但在 B4 收敛后应复查。

**已排除**（不要重走）：叠加式行分配（实测 Γ=0.34，差 3 倍；替换式确认正确，
与 §3.5.4 一致）；远场截断（上条）；尾迹 LS 本身（§9.2 精确满足）；
多值质量守恒本身（预设跳跃可精确复现，B2 gate）。

### 9.6 B3 期间交付的独立正确性修复（保留，与 B4 正交）

1. **远场 aux 必须自由**（Neumann）：把远场 aux 钉到涡的下支值会让跳跃沿尾迹
   衰减到 0（实测 0.0147→0.001），抽干环量。涡只加 **main** DOF。
   ⇒ 撤回 §5.4 选项 a 中"切单元触远场时主/辅 DOF 都要给 Dirichlet 值"的说法。
2. **尾迹必须与涡支割共面**（弦平面 y=0，design.md §4）：把 level set 沿来流
   倾斜 α 而涡支割仍水平，会在出口处留下**无尾迹承载的 Dirichlet 跳跃**，
   制造伪速度并使密度发散（实测高 M 单元全在 x≈15 远场，出现 NaN）。
3. **逐侧密度必须欠松弛**（`omega_rho`，默认 0.5）：切割带的逐侧密度会极限环，
   全量采纳（omega_rho=1）在 ~80 步后发散（Γ 0.126→0.010，M_max→6.7）。
4. **D11 后处理侧向映射是必须的**：壁面 Cp 用 `phi_main` 会让下翼面 TE 三角形
   引用 TE 的**上侧**值，cl_压力 = **−3.35**（垃圾）。按侧取 `phi_up`/`phi_lo`
   后 cl_p 与 cl_KJ 吻合到 0.4%。
5. **可压缩性由体密度承载，不由远场涡**：实测远场涡的 PG（β）缩放对 Γ **零影响**
   （软 Kutta 不传播外涡拉伸），而体密度使 Γ 0.1086→0.1256（正确方向，
   与 conforming 的 1/β 一致）。


### 9.7 ★ 最终方案（B4 交付，2026-07-12）：非线性 TE 压力相等 Kutta

**用户裁决（2026-07-12）：对称化路线不可行**——网格在 TE 处天然非对称，
不能靠配平控制体来修。因此走 §9.5 的路线 (b)：**非线性 TE 压力相等**。

#### 条件

物理 Kutta = TE 处上下表面**压力相等**（等熵下 p 是 q² 的单调函数）：

```
|q_u|² − |q_l|² = 0
```

**精确因式分解**（这是关键，使它可以每步线性化而不损失精度）：

```
|q_u|² − |q_l|² = (q_u + q_l) · (q_u − q_l) = 0
```

冻结均值 `s̄ = q_u + q_l`（上一迭代），得每步**线性**的行：

```
s̄ · (q_u − q_l) = 0
```

放在 **TE 的 aux 行**上（取代原来的下侧质量守恒）。自由流起步时
q_u = q_l = u∞ ⇒ s̄ = 2u∞，此行退化为经典线性化 Kutta。随 Picard 外迭代
重新线性化，收敛到精确的非线性条件（与密度滞后同节奏，不构成新外层循环）。

为不丢质量：原 TE aux 行的**下侧质量守恒条目改路由到 TE 主行**，主行因此承载
**上+下总质量守恒**（同时回避了"哪一侧丢方程"的任意性）。

#### ★ 为什么它不退化（而 g₂ 退化）

`q_u` 与 `q_l` 是在**两个不同的单元集合**上恢复的节点速度（TE 处的上翼面 vs
下翼面），因此 `q_u − q_l` **不是**某个跳跃场的梯度，**不会**因常数跳跃而消失。
反观 g₂：它在**同一个切单元**上收缩 ∇[φ]，常数跳跃下由单位分解恒为零（§9.2）。
**这就是全部差别。**

#### ★ 控制体必须是**壁面邻接**单元（精度的关键）

Kutta 讲的是**上下翼面**在 TE 处的速度，不是整个单元扇形的体平均。实测
（NACA0012 α=2°，不可压，vs conforming 0.1175/0.1200）：

| q 的恢复域 | coarse | medium | fine |
| --- | --- | --- | --- |
| 整个 TE 扇形（含内部/尾迹单元） | 0.1407 | 0.1355 | 0.1329 |
| **仅壁面邻接单元（含壁面 face）** | **0.1177** | **0.1191** | **0.1197** |
| conforming（同网格） | 0.1175 | 0.1200 | 0.1202 |

⇒ 壁面邻接恢复把误差从 ~11–15% 压到 **<1%**。非壁面单元（内部、尾迹）
的速度与翼面速度不同，混进平均就系统性偏高。

#### 交付结果

| --- | --- | --- | --- |

### 10.2 ★ 结论一：conforming 的 `damping_theta` **不可平移**——阻尼必须局部化

P4 的稳定器（θ·diag(A) 全场阻尼）在 B 路径上把 Γ 掐死：M0.60 无阻尼 35 步收敛
（Γ=0.0968），全场阻尼 160 步 Γ 才爬到 0.017、残差不降。**机理是结构性的**：
阻尼步的误差收缩因子 (A+D)⁻¹D 对刚度 λ 的模态为 θd/(λ+θd)——对激波的局部硬
模态 ≈ 0（有效），对光滑全局模态 ≈ 1（几乎不衰减）＝ Jacobi 光滑子的行为。
conforming 无此病，因为那里环量是**外层 secant 未知量，不在被阻尼的矩阵里**；
B 路径的隐式 Kutta 把 Γ 变成**解的模态**，同一个阻尼恰好掐住 Track B 要算的
那个量。

**修复 = 把阻尼局部化到不稳定所在处**：只对 ν>0 单元（超声速泡+激波）的节点
主行加 θ·diag，其余行不动（`damping_scope="supersonic"`，默认）。实测
M0.80 无阻尼 149 limited/M_max 50 爆掉 → 局部化 θ=0.2 后 0/0、M_max 1.5、
残差 5e-5 量级。aux 行永不阻尼（约束行对角为负，加 θ·diag 是反阻尼）。
`damping_scope="fluid"`（全场形式）保留为可复现的阴性对照。

## 11. B7：ONERA M6 三维 gate（2026-07-12，一次通过）

B1–B6 全部跑在准二维网格上。准二维尾迹面是一条平直窄带：**无后掠、无翼尖、无展向**。
因此 LS 路径有三块机械从未被真正测试过——B7 就是为它们存在的：

1. **TE 折线直纹 level set（D9）**：尾迹面在后掠的 root→tip TE 折线之间直纹生成，
   其分段标架 (v, d̂, n̂) 是**斜交**的（展向轴与尾迹方向不垂直）。B1 已在此处发现并
   修复过真实缺陷（正交投影把下游距离泄漏进展向坐标，误裁掉约 60% 的真实切割集）。
2. **展向裁剪** `0 ≤ q ≤ span_length`：它才是"Γ 在翼尖离散地归零"的机制——尾迹**平面**
   穿过但位于尾迹面**展向之外**的单元被拒绝（`CutElementMap.beyond_tip_elems`），
   那里不承载任何跳跃。这是 conforming 路径自由边规则（Γ(tip)=0）的 level-set 对应物。
3. **g₂ 分量的展向跳跃梯度自由**——尾涡自由度（D1）。准二维网格上根本没有"展向"可自由。

### 11.1 结论：三维专属机械**无需新求解器代码**

B1 的两处三维修复（斜交标架 + 展向裁剪）直接生效：双网格上 Γ(z) 从翼根单调衰减，
**翼尖 Γ ≈ 3e-4**（离散归零）。唯一的代码缺口在**后处理**——TE 节点是多值的，
而 `section_cp_curve` 只接受单个节点场。补上两个函数（`post/surface_ls.py`）：

* `section_cp_curve_levelset`：镜像 `section_cut._wall_section_points` 的平面切割循环，
  但每个壁面三角形按 **D11 规则**（外法向升力分量符号）从 `mvop.side_potentials` 里取
  自己那一侧的梯度。**上表面与 `main` 场结果逐位相同**（上表面三角形的 TE 顶点本来就
  读 main = 自己侧），所以 gate 的激波指标（全在上表面）不依赖新机械；下表面才是
  D11 真正吃劲的地方（用 main 读会得到 B3 记录的 cl_pressure = −3.35 垃圾值）。
* `cl_pressure_3d_levelset`：按**平面投影面积** s_ref 归一（`cl_pressure_levelset` 用的是
  准二维展向厚度 dz），与 `cl_kj_3d` 配对做 V6 一致性交叉校验。

**代价远低于计划中的风险估计**：~12k 三维 DOF 上每个 outer 的 `spsolve` 约 0.6 s，
七级 Mach 续接整体 ~20 min（不是"数小时"）。

### 11.2 ★ 结论一：B6 的**升力反转**在三维一次复现

M∞0.84 / α3.06 / coarse / `farfield="neumann"` / ramp 0.60→0.84 @ dm 0.04：

| | M1 嵌入 | M4 无尾迹 | P5 conforming **Picard** | P8 conforming **Newton** |
|---|---|---|---|---|
| cl_KJ | 0.2765（**+2.7%**） | 0.2710（**+0.7%**） | 0.24788（**−8.6%**） | **0.2692**（真解） |
| V6 | 1.77% | 1.97% | 2.40% | — |
| 激波 η .44/.65/.90 | 0.635/0.588/0.449 | 0.634/0.584/0.454 | 0.596/0.570/0.425 | 0.596/0.541/0.362 |
| Γ 根→尖 | 0.1076 → −0.0003 | 0.1055 → +0.0003 | 0.097 → 0.0206 | — |
| M_max, lim/flr | 1.453, **0/0** | 1.368, **0/0** | 1.398, 0/0 | 2.13 |

以 conforming **Newton 真解**为基准（B6 的用户裁决基线，非 conforming Picard）：
**LS Picard 落在 +2.7%(M1) / +0.7%(M4)，而 conforming Picard 低了 −8.6%**。机制与
§10.4 在二维测到的完全一致：LS 路径**没有可以提前停的 Γ 外循环**（隐式 Kutta 使 Γ
成为**解模式**，与场一同收敛到残差 ~1e-6），而 conforming Picard 的冻结-Γ 内解 +
预算截断的逐站 secant 会**欠环量**（P4-erratum 偏差；P8 独立测得 M6 medium +7.9%）。

⇒ **若按 P5 设 gate，等于因为 B 路径"更接近真解"而扣它的分**——所以 B7 的升力锚定在
Newton 上。并且**无尾迹的工作流网格 M4 是两者中更准的那个**，这正是 Track B 的目的。

### 11.3 ★ 结论二：三维远场 = `neumann`；P5 的 Γ(z) taper 在 B 路径上**结构性地不需要**

B 路径的涡远场（`picard_ls._farfield_main`）是**展向均匀**的二维点涡，其支割线是
**每个 z 上都相同**的射线 y=0, x>0。三维机翼上它有两个**互相独立**的错法，两者都表现为
**尾迹面离开计算域的出口处**出现伪近声速斑（下表为 M∞0.5 时出口处最大局部 Mach）：

| 远场 | 出口 M_max | 机制 |
|---|---|---|
| `neumann`（无涡） | **0.513** | — |
| vortex，尾迹面重新对准 y=0（共面） | 0.825 | 仅剩 (b) |
| vortex，尾迹面按 α 对准（默认） | **0.958** | (a) + (b) |

* **(a) 非共面**：尾迹面按 (cos α, sin α, 0) 对准，到出口 (x≈10c) 已抬升到
  y ≈ x·tan α ≈ 0.5，远离涡的 y=0 支割线 ⇒ 出口承载一个**没有任何切割去支撑的**
  Dirichlet Γ 跳跃。这正是 B3 记录的 load-bearing 共面规则，在三维上的形态。
* **(b) 展向均匀**：即使把尾迹面重新对准共面，伪迹**缩小但不消失**——一个标量 Γ 无法
  匹配 Γ(z)→0，而翼尖之外**根本没有切割**。这就是 P5 的 branch-ray 伪迹，conforming
  路径当年的解药是 Γ(z) taper（`farfield_dirichlet(spanwise_gamma=True)`）。

⇒ **neumann 不带涡，因此两个缺陷都不可能存在**：Γ(z) taper 在 B 路径上不是"尚未实现"，
而是**结构性地不需要**。代价是 B5 测得的 O(Γ/R) 出口截断（紧凑域上几个百分点的升力），
这也是 gate 采用 A/B 带而非 <1% 带的原因。（与 §10.3 一致：折叠区 live Γ→涡 反馈增益 >1。）

### 11.4 ★ 结论三（计划外）：Γ(z) **本身就是展向光滑的**——无需任何平滑

把**真实的** P5 Γ(z) 曲线叠上去（`gamma_of_z.png`）的那一刻就看见了：conforming 的
Γ(z) 有明显的**逐站抖动**，而 LS 的两条曲线是光滑的。量化（Γ(z) 的二阶差分 RMS，
按曲线自身幅值归一）：

| | 归一化粗糙度 |
|---|---|
| LS，M1 嵌入 | **0.0079** |
| LS，M4 无尾迹 | **0.0091** |
| conforming P5 | **0.0970**（**粗糙 11–12 倍**） |

原因是结构性的：conforming 路径**每个 TE 站点解一个独立的 secant**，于是 Γ(z) 携带
逐站噪声——这正是 P5 的 `INVESTIGATION_gamma_smoothing.md` 追过的缺陷（结论：展向 Γ
**平滑**会把 Γ 推**离**自洽值，A–E 五条路线全部作废），也正是**单站失效**（st133 欠环量
32%）耗掉 P5 整整一次调查的那套机械。而隐式 Kutta **根本没有逐站循环可供产生噪声**：
Γ 是耦合系统的**一个解模态**。

⇒ Track B 不是"修好"了 P5 的展向 Γ 问题，而是让它**结构上不可能发生**。

### 11.5 诚实缺口（已记录，本阶段不追）

1. **收敛语义 = P4/B6 记录的 Picard 残差尾，不是 `tol_residual`。** 顶部若干 Mach 级
   耗尽 600-outer 预算，停在 |R| ~4–6e-6（M1：0.72–0.84 级；M4：0.68/0.76/0.84 级）。
   但**每一级都有界且物理**（全程 0 limited / 0 floored），全部 gate 指标在带内 ⇒
   gate 断言的是**有界+在带**，而非 `converged`。解药是 LS Newton。
2. **LS Newton 在 M6 上推迟**：`solve/newton_ls.py` 原用朴素 `splu`，而 P8/N6 实测
   真三维 LU 填充约为 2.5D 的 ~100×。**★ B11（2026-07-14）部分解决**：`newton_ls`
   现有 `precond="ilu"|"amg"` 迭代逃逸（§5.3 落地），提供了一条非 splu 路线；但
   **lagged-LU（`direct_refactor_every`,直接因子重用）移植仍未做**,留作后续项——
   若 M6 证据显示 GMRES+AMG 够用则按"被取代"关闭。折叠能力 B6 已在二维证明。
3. **激波位置比 P5 靠后 0.02–0.04c**（在 ±0.06 带内，且与更高的环量自洽）；对 P8
   **Newton** 激波，η=0.90 站靠后 0.087——记录在案：与 Newton 做同类比较需要先有 LS Newton。
4. **仅 coarse**（medium/fine M6 推迟：成本 + 折叠风险）。

证据：demo `cases/demo/b7_onera_m6/`（**35/35 PASS**，4 图 + summary/farfield/checks CSV），
`tests/test_b7_onera_m6.py`（6 快速 + 5 gated）。conforming 路径逐字节未动。

## 14. B15:LS Newton 跨声速续接 + N5 冻结选择（2026-07-15,已关闭 — GB15.1–15.4 全绿,见 roadmap B15 条目）

### 14.0 动机:24–38 min 的成本在**哪里**

M6 medium M0.84 的 LS 解耗时 **2304.7 s(wake-free)/1469.9 s(嵌入)**
（`cases/demo/m6_medium_ls_workflow/`）。逐级缓存显示:7 级 Mach ramp
（0.60→0.84,dm 0.04）中 **0.80 与 0.84 两级不收敛**,各自烧满 200 outer 预算,
停在**激波位置残差 plateau**（P4/B6/N5 软模）。`tol_residual` 已被设到 1e-5
（plateau 之上）——设 1e-7 会让每级烧满预算（~1 h）。**这是 Picard 方法的固有
属性,不是调参问题。** Newton 没有这个软模。

但 B15 之前 `newton_ls` 跑不了 ramp:`freeze=` 是**无实现的保留字段**;收敛判据
硬性要求 0 limited/floored（激波限速单元恰好挡住它）;**没有 Mach ramp 包装器**。

### 14.1 逐侧冻结:内核**原样复用**,只是接线

`kernels/upwind.py` 的冻结机械（`classify_upwind_branches`、
`rho_tilde_frozen_sweep`、`rho_tilde_frozen_sensitivities_sweep` +
`UpwindOperator.freeze_upwind_state/rho_tilde_frozen/…`）**逐字复用**:LS 的逐侧
算子（`multivalued.py::_side_upwind`）本就是 **walk 模式**的 `UpwindOperator`,
只是把 face graph 掩到自己一侧。所以 B15 = 接线,不是新数值。

新增:`MultivaluedOperator.newton_side_data(frozen=…)` +
`freeze_side_state`（逐侧 `(upstream, branch)` 捕获）。冻结态下 `n_floored` 只数
`branch==3`（分支 0–2 **不重新施加 floor**）⇒ **干净冻结天然 0 floored**,这正是
解开 0-clamped 收敛判据的机制。限速器（m_cap）保持 **live**（与 conforming 一致）。

**`LSNewtonSystem`（新类）:** 把残差与 Jacobian 装配抽出为**单一代码路径**,求解
器与 FD gate 共用——Jacobian 必须是求解器**实际所用**残差的导数,一个自己重写装配
的 FD 测试只是在测它自己的副本。

**GB15.1 FD:** rel **6.7e-9**（eps 1e-5;1e-6/1e-7 给 5.8e-8/6.0e-7,舍入主导的
干净标度 ⇒ 确系真导数),ε-guard 仅排除 3.1% 的行,真实激波区（nu_max 0.785,
1118 个 branch-1/2 单元）。冻结 sweep 在冻结点**逐位复现** live 密度。

### 14.2 ★ 触发器勘误:conforming 的配方**不可平移**

`solve/newton.py` 的冻结触发是 `(r < freeze_tol) OR live_stalled`。**逐字移植会
坏事。**

**实测（NACA medium M0-embedded M0.75）:** LS 的 live 残差在 ±2× 区间**震荡数十
步但仍在下降**（同期 γ 从 0.183 走到 0.243——这是**刚性方向上的慢进展,不是
stall**）。`live_stalled` 在 ~5e-6 就误触发,冻结了一个**仍在移动**的迎风指派;
冻结步随即发散 → revert → 重新武装 → 再发散:**3 次 revert,不收敛**——而**原封
不动的 live 路径 54 步收敛到 7.5e-12**。

去掉 stall 触发后,同一算例在 |R|<1e-6（指派已安定）才冻结,**53 步收敛到
2.1e-12,0 revert,落在与 live 完全相同的 γ=0.243305**,且 `residual_unfrozen`
= 2.1e-12（LIVE 残差在冻结解处同样紧）⇒ **诚实通过**。

⇒ **LS 路径的冻结只由 `freeze_tol` 武装。** `stall_window` 只留给（opt-in 的）
`accept_on_stall` 松接受路径。

### 14.3 ★ 冻结确实治愈了一个**真实的极限环**（GB15.2）

**NACA coarse M0.75:** live LS Newton **不收敛**——停在一个**周期 6 的极限环**
（3.2e-7, 2.8e-7, 2.7e-7, 1.3e-6, 8.6e-7, 4.3e-7,精确重复）、|R|≈2.7e-7（高出
tol 三个量级）、**0 limited/floored**（干净停滞 = 指派 churn）。武装冻结后:
**22 步收敛到 8.5e-13**,0 revert,**γ 0.218809 vs live 环的 0.218804** ⇒ 冻结
**只去掉 churn,不改变解**。

**安全网（新增,B15）:** `freeze_max_reverts`（默认 3）——反复发散的冻结会**永久
解除武装**并回落到 live 路径 ⇒ 冻结**只可能帮忙,绝不会赔掉收敛性**。报告的
`n_limited/n_floored` **总是 live 重读**:冻结收尾天生 0 floored,不能拿它自证。

### 14.4 Mach ramp 包装器 `solve_multivalued_newton_transonic`（GB15.3）

镜像 conforming N5/N6:上行 `mach_schedule`;**只从收敛级热启动**;失败级把 dm
折半、在其**下方**插入新级且**严格**运行;可选 `upwind_c_post`(López 2.0→1.6);
诚实的 `target_reached`/`m_final`（P13/G13.3 勘误:**绝不对未达标 ramp 的状态做
统计**）。

**★ `intermediate_tol` 的 LS 专属差异:中间级**保持冻结武装**。** conforming 的
松掩码（`newton.py:888`）在中间级设 `freeze_tol=None`;LS 不能这么做——接受判据
的**每一条**路径（含松路径）都要求 0 limited/floored,而 0.60→0.84 的 ramp 里
**激波在中途成形**,这些中间级带着限速单元,只有**通过冻结**才能拿到 0-clamped
接受。⇒ **松开停止容差,保留机制。** conforming 折叠区的禁忌（松级留下一个未被
跟踪的 **Γ 种子**,G10.2）在此**无对应物**:LS 无 Γ 自由度,Γ 是 `phi_ext` 内的
解模态。

**★ 种子也必须继承 B13 lagged-LU（实测陷阱）:** `_seed_from_picard` 原先不转发
`direct_refactor_every`,于是种子的**每个 outer 都做一次完整稀疏分解**——M6
medium 上 ≈17 s × n_seed,即 40-outer 种子 **~11 min 的纯热启动**,把它所要热启动
的 Newton 解本身都盖过去了。种子沿用 lifting 的 `direct_reuse_rtol=1e-10`（**不是**
Newton 的 1e-8):Picard 不动点只由其 lag 容差钉住,不精确的 reuse 步会**移动**它
的停止点（B13,`picard_ls.py:361-366`）。

**GB15.3 实测（NACA coarse M0.80 / α1.25 = B6 gate 工况;同网格 conforming-Newton
真值 shock 0.658 / cl_p 0.459 / M_max 1.408）:**

| 方法 | 墙钟 | γ | \|R\| | 收敛级 | 迭代 |
|---|---|---|---|---|---|
| Picard | 43.3 s | 0.190374 | **1.55e-5**(停滞) | **3/5** | 962 outer |
| Newton 严格 | **8.3 s (5.2×)** | 0.212445 | **3.1e-12** | 5/5 | 48 步 |

## 16. B16:LS Newton 远场 BC 通用化——远场 aux DOF 钉扎（2026-07-17，已关闭）

### 16.1 根因（GB16.1，实测；修正了预注册提案的机理）

尾迹水平集**没有出流裁剪**（`cut_elements.py`：一个交叉只需在 TE 下游且在展向范围内），所以
sheet 抵达远场边界，被它切到的外区节点各持一个 aux DOF，而这些 aux **只受一个巨型外区单元上的
wake-LS 行约束——该行近奇异**。在 freestream Picard 收敛态处，这些 aux 携带垃圾值
（coarse 翼身：x≥10 处 |jump| **53.4** vs 物理 Γ̄ **0.0586**）；Picard 不动点把这些行连同垃圾
一起解为零（自洽），但 Newton 残差把它读成 O(1) 不一致——**恰好是 8 个远场 MAIN 行，
max|R| = 84.457**（逐位复现）。aux 块的条件数是单数字铁证：`jaa_diagnostic` cond1 =
**9.1e18**（legacy 自由 aux——**高于 GB14.1 的 1e14 上限**，即真奇异；勘误 2026-07-19：
旧文引的 6.36e18 是 CSV 前预跑值，committed CSV 两个 epoch 均为 9.1e18；onenormest
估计随运行漂移，要点是跨上限的 12 个量级落差）→ **8.70e6**（钉扎后）。

⚠ **提案的机理是错的**：它把 Picard 的成功归于 `closure="continuity"`（weld）对 Newton 的
`wake_ls`，但升力/跨声速 Picard **也用** `wake_ls`（weld 只在 Laplace 种子）——两路 aux 行相同，
差别在于 Picard 不动点吸收近奇异行、Newton 残差不吸收，**不是 closure 的差别**。

### 16.2 修复：`farfield_aux="pin"`（默认），按模式适配（用户裁决）

在 Dirichlet 远场（freestream/vortex，远场 MAIN DOF 已是 Dirichlet）上，把远场**边界** aux 也
放进 Dirichlet 集，值取其宿主节点所在支：

- **freestream**：aux = 与宿主相同的单值 φ∞ ⇒ 环上 jump→0。这与 freestream BC 本身在 25-MAC
  边界抹掉环量一致（O(Γ) 的环局部误差，与该 BC 的截断同级；GB16.4 是它的检测器）。
- **vortex**：aux = `main − side·γ`，每步随 γ 刷新 ⇒ jump→γ。`side = cm.node_side ∈ {±1}`
  （节点被 ε 移位后永不为 0），是 conforming `lower_branch_mask` 的类比。**与 B3 的负结果不同构**：
  B3 把两侧都钉到一支（jump→0）抽干了环量；这里分侧、保 jump=γ。

**`neumann` 逐位不变**（其外区 aux 本就受 wake-LS 行约束，且所有 committed LS-Newton 锚都用

## 17. B17：远场 aux 钉扎必须携带 jump=γ 而非 0（解决 GB16.4，2026-07-18，已关闭）

**GB16.4 不是非收敛，是 B16 freestream pin 的边界条件建模错误。**

### 17.1 机理

尾迹在真问题里把势跳跃 [φ]=Γ 沿尾迹面一路带到出流边界（无穷远）。B16 的 pin 在
`farfield="freestream"` 上把出流环上的 aux 钉到 host 的单值 φ∞，即 **jump→0**——这等于在
出流处**杀死尾迹环量**，是 O(Γ) 的**主阶**误差（Γ≈0.066，却移动升力 22%），不是 design §16.2
声称的"受控 O(Γ) 环误差"。

- **legacy**（aux 自由，由 wake-LS 行约束）尝试沿尾迹维持跳跃：medium 大致成功（远场 |jump|≈0.105≈Γ
  ⇒ cl_p 0.2165≈conforming），coarse 巨型外层 tet 近奇异 ⇒ 垃圾 |jump|=53 ⇒ cl_p 0.1853。
- **pin jump=0**：medium 掉 22%（0.2165→0.1690）；coarse "蒙对"（0.1853→0.2086）纯属 jump=0 的误差
  恰好抵消了 legacy 的 +53 外层 tet 垃圾。

### 17.2 判别性证据（两个求解器同 BC 一致）

给 Picard 驱动（`solve_multivalued_lifting`）加同样的 freestream pin（新 `farfield_aux` 旋钮），
medium Picard-pin 干净收敛（res 7.5e-8，34 outer）到 **cl_p 0.1691**，与"停滞"的 Newton-pin
**0.1690** 吻合 0.1%。两个完全独立的求解器落到同一个不动点 ⇒ 那是一个真实的 BC 决定的态，**不是**
Newton 停滞。B16 的可能性 (a) 被否证——pin 不动点本身就是错的。

后处理排查（GB17.2）：cl_p（壁面压力面积分）与 cl_KJ（环量积分）**同步移动**（medium 两者都 ~22%）
⇒ 是真实流场态变化，非后处理伪影。用户观察的"Cp 目测对齐却 cl_p 差 22%"是 Cp 轴尺度错觉——绘制的
展向 sectional cl 是 **Γ 基** `2Γ/(u·c)`，逐站 ∫Cp 实差 24–44%，而 Cp 曲线绝对差仅 ~0.03–0.05。

### 17.3 修复：`farfield_aux="pin_gamma"`（jump→γ，新默认）

出流 aux = host φ∞ − side·γ，随迭代刷新当前 γ（Picard 每 outer；Newton 每步，冻结于步内）。这是
B16 正确识别的近奇异外层 aux 的**同一个** Dirichlet 条件数治疗（cond1 O(1e19)→8.7e6，与 jump 值
无关），但用了**物理正确的环值**。三角单调闭合：

| 远场 aux | coarse cl_p | medium cl_p | 趋势 |
| --- | --- | --- | --- |
| conforming（P14 Newton，参考） | 0.2089 | 0.2173 | ↑ |
| legacy（自由 aux） | 0.1853 | 0.2165 | coarse 被 \|jump\|=53 外层 tet 污染 |
| pin jump=0（B16） | 0.2086 | 0.1690 | ✗ 非单调、杀出流环量 |
| **pin_gamma（B17）** | **0.2087** | **0.2117**（Picard）/ **0.2115**（Newton） | ✓ 单调 |

Newton-pin_gamma 与 Picard-pin_gamma 两分辨率都吻合 0.1%，均欠 conforming 0.1%/2.6%（远场截断）。
★ **B16 混淆了两个正交问题**：远场近奇异**条件数**（pin 治，jump 值无关）与出流**环量**（需 jump=γ）。
medium Newton-pin_gamma 仍带翼身交界 churn（nlim 42/nflr 40，res 5.5e-5，**G1.6/GB9.4** 类）——但
γ 稳定 0.06420、cl_p 0.2115 已正确，churn 只限制残差地板不再污染升力（第三个、既有的问题）。

### 17.4 默认值与惰性（用户裁决 2026-07-18）

`farfield_aux="pin_gamma"` 成两个求解器新默认。**仅作用于** `farfield="freestream"`；在 vortex/neumann
上**惰性**（逐位等同 legacy）——所以所有已提交的 2.5D NACA vortex/neumann Picard 运行、所有 neumann
Newton 锚点逐字节不变。B9/B16 的 freestream Picard demo 已钉 explicit `farfield_aux="legacy"` 以保留
committed 数字；B16 的 jump=0 数字用 explicit `"pin"` 复现。`"pin"`（jump=0）保留为诊断值。

### 17.5 vortex 评估（GB17.6，RECORDED，用户要求）

`farfield="vortex"`（物理一致的升力远场：freestream + PG 涡加在远场 MAIN dof）**不能**消除 pin_gamma
的 medium 2.6% 余差——它从**另一侧 bracket** conforming（medium **+2.5%**，vs pin_gamma −2.6%），且其
自由远场 aux 在 **coarse 上 churn**（res 3.2，|jump|=71，需自己的 pin）。2–3% 是远场截断，非 bug；
**freestream pin_gamma 仍是推荐 BC**（两分辨率都干净收敛）。

证据：`tests/test_b17_farfield_pin_gamma.py`（6，ungated）；demo `cases/demo/b17_farfield_pin_gamma/`
（Part 1–2 coarse ungated + Part 3 medium 门控）。求解器仅动 `newton_ls.py`/`picard_ls.py` 的 freestream
远场 aux 接线（vortex/neumann/conforming 路逐位不变）。

## 19. B19：LS Newton Jacobian 在 3-D 的精确化（2026-07-18，已关闭）

Terms 2/3（密度敏感度块，§5.5）的**行与列来自不同映射**：行 = 质量守恒散射映射
`dofvec`（残差落点），列 = `side_potentials` 的逐节点**读取**映射 `readvec`
（`MultivaluedOperator._side_readvecs`）。两者在 cut 元素上恒等（已断言），
pre-B20 在 mixed-side plain 元素上分歧。同一二元性下沉一层：残差为
`ρ̃(读取场梯度)·V·(散射场梯度·B_a)`，故行因子用 `grad_row`（散射场）、列因子用侧场
梯度。两处缺项都修（targeted 探针 1.146e-01 → 1.334e-08，ε 判别量由 ε 无关翻转为
~1/ε）；R 逐位不变 ⇒ 无任何已收敛结果移动，也无收敛收益（GB19.4 阴性——平台是
B15 选择 churn 极限环）。盲区不变量（勘误后）：quasi-2D 有 129 个 mixed-side plain
元素、**0 个接触 cut 节点**（"无该类元素"是错的说法）。测试
`tests/test_b19_jacobian_3d.py`。

## 20. B20：mixed-side plain 元素的密度来源 = MAIN 场（2026-07-18 永久采纳；离散规范变更）

**这是 shipped 离散的一部分（无开关）**：uncut 但四节点跨越 level set 的元素
（mixed-side plain 类，`_mixed_plain_mask`）在 `newton_side_data` /
`element_densities` 的密度路径中，其梯度与 q² 一律以 **MAIN（单值）场**替换
（`_apply_main_density`）——理由：该元素的刚度与 MAIN 场收缩（散射在 main DOF 上），
密度若读 side 场（在 cut-shared 节点代入对侧 aux 值）即"一个方程、两个速度场"的
内部不自洽，且实测制造虚假超声速态（GB19.6：q² 3.2229 vs 1.3379，ρ 误差至 45.3%）。
一致性推论：readvec/grad 列因子在该类元素上回到 main（§19 的映射随之在全部四类
元素上与散射映射重合）。2.5-D 逐位不变（该类 0 接触 cut 节点）；3-D 已提交 LS
数字重基线（2026-07-19）。工作区别名陷阱：`PicardOperator.velocities` 返回共享
缓冲视图——密度路径内重算 main 梯度前必须 `.copy()`。

## 21. B21：freeze 捕获与 live 密度路径的一致性（N1，2026-07-19，已关闭）
